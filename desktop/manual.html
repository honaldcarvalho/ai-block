Manual Oficial de Instalação e Uso

Sobre

O que é o AI-Block?

O AI-Block é uma ferramenta de segurança desenvolvida em C que protege a sua máquina contra acessos a dezenas de Inteligências Artificiais e rastreadores (como ChatGPT, Claude, Gemini, e IAs Jurídicas Brasileiras). Como todos os dados sensíveis ficam protegidos no inovador Banco de Dados SQLite Zero-Dependências, ninguém pode simplesmente abrir um arquivo de texto para burlar suas configurações de bloqueio sem possuir a Senha Mestra.

Configuração

Como Instalar e Iniciar

No Linux (Debian, Ubuntu, Kali)

1. Rode o instalador: sudo ./install_linux.sh

2. **Menu de Aplicativos**: Procure por "AI Block" no seu menu. Você pode abri-lo sem terminal aberto!

No Windows

1. Rode install_windows.bat como **Administrador**.

2. Um atalho com ícone personalizado aparecerá na sua Área de Trabalho.

Blindagem Automática (DoH)

O instalador agora configura automaticamente políticas para **Chrome, Edge e Firefox**, garantindo que o bloqueio via hosts seja sempre respeitado.

Segurança

Passo A: A Senha Mestra

Após instalar, abra o aplicativo pelo Menu ou Área de Trabalho. Na primeira execução, ele solicitará a criação de uma **Senha Mestra**. Esta senha protege o acesso à interface e impede que usuários não autorizados alterem a lista de bloqueio.

Privilégios Elevados: No Linux, a interface abre como usuário comum. Ao clicar em "Bloquear", o sistema pedirá a sua senha de administrador apenas para salvar as alterações com segurança.

Interface e Ferramentas

Bloqueio Inteligente

O AI-Block não apenas bloqueia os sites, mas também fecha as brechas onde eles costumam escapar:

  • IPv6 Completo: Bloqueia endereços via IPv4 e IPv6 (::1).
  • Subdomínios Automáticos: Cada site bloqueado (ex: google.com) tem sua versão www. bloqueada na hora.
  • CLI Profissional: Comandos como --block, --add e --update continuam disponíveis para usuários avançados.
