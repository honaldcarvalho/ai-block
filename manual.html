Manual Oficial de Instalação e Uso

Sobre

O que é o AI-Block?

O AI-Block é uma ferramenta de segurança desenvolvida em C que protege a sua máquina contra acessos a dezenas de Inteligências Artificiais e rastreadores (como ChatGPT, Claude, Gemini, e IAs Jurídicas Brasileiras). Como todos os dados sensíveis ficam protegidos no inovador Banco de Dados SQLite Zero-Dependências, ninguém pode simplesmente abrir um arquivo de texto para burlar suas configurações de bloqueio sem possuir a Senha Mestra.

Configuração

Como Instalar e Iniciar

No Linux (Debian, Ubuntu, Kali)

1. Abra o terminal na pasta do projeto e rode o instalador:

sudo ./install_linux.sh

Ele organizará tudo em /opt/ai-block/ e criará o comando global.

No Windows

1. Rode o arquivo install_windows.bat como Administrador.

Ele levará o projeto inteiro (incluindo este manual e o .exe) para C:\Program Files\ai-block\ e baixará a lista oficial de sites bloqueados de imediato.

Segurança

Passo A: A Senha Mestra

Após instalar, é obrigatório registrar a sua senha de governança. Para isso, rode o aplicativo limpo pela primeira vez como Administrador/Root:

sudo ai_block

Isso fará com que o sistema crie o banco .db criptografado no sistema e solicite a criação da sua chave mestra a ser guardada de forma segura.

Interface e Ferramentas

Interface Gráfica vs Linha de Comando

O AI-Block permite que você opere via cliques pela bela UI em GTK3, ou via terminal (focado em execução automatizada e eficiência).

Comandos do Terminal Automáticos

Todos estes comandos exigirão a entrada da sua senha mestra no console:

  • --block: Habilita permanentemente o bloqueador de IA (Joga as regras de bloqueio no arquivo /hosts da sua máquina).
  • --unblock: Limpa perfeitamente *apenas* os bloqueios de IAs que fizemos do arquivo de rotas local.
  • --add seudominio.com: Registra permanentemente um novo domínio proibido em seu banco de dados interno.
  • --update: Se conecta aos servidores oficiais (Croacworks) para baixar sigilosamente nossa lista mestra mais recente e emagá-la com o Banco de dados existente!
  • --import_json [arquivo]: Funcionalidade bônus para caso você deseje subir uma lista sua em JSON estruturado com chave "domains".
